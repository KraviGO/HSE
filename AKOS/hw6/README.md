## Выполненные скрипты

### 1. check.sh
- **Назначение**: Проверка введенного числа
- **Используемые конструкции**: `if`, `elif`, `else`
- **Функционал**: Определяет знак числа и четность

### 2. counter.sh  
- **Назначение**: Счетчик повторений
- **Используемые конструкции**: `while`
- **Функционал**: Выводит "Hello World" заданное количество раз

### 3. functions.sh
- **Назначение**: Демонстрация работы функций
- **Используемые конструкции**: функции, вызов функций из функций
- **Функционал**: математические операции

### 4. caller.sh
- **Назначение**: Вызов других скриптов
- **Используемые конструкции**: вызов внешних скриптов
- **Функционал**: Последовательно запускает check.sh и counter.sh

### 5. file_manager.sh
- **Назначение**: Работа с файлами
- **Используемые конструкции**: `if`, проверка существования файлов
- **Функционал**: Показывает файлы в папке и проверяет существование указанного файла
